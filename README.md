# AI-Based Stock Market Prediction

## Setup
1. Create virtual env and activate.
2. `pip install -r requirements.txt`

## Quick start
`python main.py`

## Structure
- `data/` - raw & processed data
- `src/` - code modules
- `notebooks/` - EDA & experiments
- `models/` - saved models
